# ProvenanceIP GitHub Pages React Site

This folder is ready to upload to GitHub and publish with GitHub Pages.

## How to publish

1. Create a new GitHub repository.
2. Upload all files and folders from this project, including `.github`, `src`, `index.html`, `package.json`, and `vite.config.js`.
3. Go to the repository's **Settings**.
4. Go to **Pages**.
5. Under **Build and deployment**, choose **GitHub Actions**.
6. Go to the **Actions** tab and wait for the deploy workflow to finish.
7. Your live link will appear in **Settings > Pages**.

## Local preview

Run these commands in the project folder:

```bash
npm install
npm run dev
```
